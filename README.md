## 캡션 스터디 소개

캡션 스터디는 자신이 보고싶은 영상을 봄과 동시에 외국어 공부도 할 수 있는 어플리케이션입니다.

## 캡션 스터디 사용 가이드

1. 보고싶은 영상에 들어가서 "공유" 버튼을 누르고 "링크 복사" 버튼을 누르기
